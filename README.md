# Ohmyfood - Projet 4 (Formation Intégrateur Web)

Ohmyfood : Il s’agit d’une jeune startup qui voudrait s'imposer sur le marché de la restauration. 
**Mission** : Développer un site mobile-first qui répertorie les menus de restaurants gastronomiques avec des animations CSS et de structurer le code via le préprocesseur SASS.

## Organisation 
  - Architecture modulaire simplifié utilisant le **pattern 7-1**.

## Fonctionnalités
**Mobile-first** : Design conçu pour mobile, adapté ensuite sur tablette et desktop.
**Animations CSS** :
  - Loading Spinner au chargement de la page.
  - Remplissage progressif des icônes "cœur" au survol.
  - Apparition progressive des menus par catégorie.
  - Animation de validation (Check) lors de la sélection d'un plat.

## Technologies utilisées
  - HTML
  - CSS (via SASS)
  - FontAwesome
  - Google Fonts (Roboto, Shrikhand)
